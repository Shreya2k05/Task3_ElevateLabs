## üß™ Flexbox Properties Used
* `display: flex`: Established the flex context for containers.
* `flex-direction`: Controlled the primary axis of the layout.
* `justify-content`: Managed horizontal spacing (e.g., `space-around`, `center`).
* `align-items`: Handled vertical centering of elements.
* `flex-wrap: wrap`: Prevented overflow by moving items to the next line on small screens.
